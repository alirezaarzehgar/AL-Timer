## Introdoction for AL-Embedder
	this program can record time

## Dependencies
	- libgtk+-2.0

## Installing
	$ git clone https://github.com/alirezaarzehgar/AL-Timer.git
	$ cd AL-Timer
	$ make
	$ ./AL-Timer

## using
	1- ./AL-Timer
	2- you can select {stop} {start} {refresh} {get value} button
